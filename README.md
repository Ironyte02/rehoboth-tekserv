# Rehoboth Tekserv — Landing Page

Single-scroll marketing site for **Rehoboth Tekserv** — *Marine, Offshore & Hazardous Area Electrical Inspection and Compliance Services.*

> Reliable Partner · Safe Operations · Complete Compliance — *Anchored in Excellence.*

## Overview

A self-contained, single-file landing page (`index.html`) with no build step. Brand banner and logo are used as-is in `assets/`.

### Sections
- Sticky navbar with brand logo
- Banner header (brand artwork) + navy intro/CTA band with rotating compass
- About
- Services (9 specialist service lines)
- Industries served (Marine, Offshore, Oil & Gas, Petrochemical)
- Why Us + animated stats
- Call-to-action band
- Contact footer

### Highlights
- Pure HTML/CSS/JS — open `index.html` in any browser
- Brand palette (navy / gold / sea blue) throughout
- Content-relevant motion: rotating compass, 3D-tilt service cards, scroll reveals, count-up stats
- Respects `prefers-reduced-motion`
- Responsive (desktop / tablet / mobile)

## Run locally

Just open `index.html`, or serve the folder:

```bash
python -m http.server 4178
# visit http://localhost:4178
```

## Assets
- `assets/banner.png` — header banner
- `assets/logo.jpeg` — full brand logo
- `assets/emblem.png` — transparent emblem (navbar/footer)
